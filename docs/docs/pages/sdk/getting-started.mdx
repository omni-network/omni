# Getting Started

The Omni SDK contains two main TypeScript packages available on the [npm registry](https://www.npmjs.com/):

- [`react`](/sdk/getting-started/react.mdx) for client-side React applications.
- [`core`](/sdk/getting-started/core.mdx) for any supported JavaScript runtime.
